1.1 Conjuntos\textsf{\textbf{\color{c77b46}{1.1} \color{#d4d2cc}{Conjuntos}}}1.1 Conjuntos﻿

1.1.1 ¿Queˊ es un conjunto?\textsf{\textbf{\color{c77b46}{1.1.1} \color{#d4d2cc}{¿Qué es un conjunto?}}}1.1.1 ¿Queˊ es un conjunto?﻿

Un conjunto\small \color{c77b46} \textsf{\textbf{conjunto}}conjunto﻿ es una colección finita o infinita de objetos o elementos concretos o abstractos, representados con letras mayúsculas (AAA﻿, BBB﻿, CCC﻿). Los objetos o elementos pertenecientes se identifican con letras minúsculas (aaa﻿, bbb﻿, ccc﻿). Cuando un elemento xxx﻿ pertenece a un conjunto AAA﻿ escribimos x∈A\bm{\color{c77b46} x \in A}x∈A﻿ (de lo contrario, notamos x∉A\bm{\color{c77b46}x \notin A}x∈/A﻿).

Un subconjunto\small \color{c77b46} \textsf{\textbf{subconjunto}}subconjunto﻿ es un grupo de elementos dentro de un conjunto dado. Si BBB﻿ es un subconjunto de AAA﻿, todos los elementos del primero son elementos de este último (BBB﻿ está contenido o incluido en AAA﻿), y se escribe B⊆A\bm{\color{c77b46}B \subseteq A}B⊆A﻿. Para que B⊈AB \nsubseteq AB⊈A﻿, basta con que un elemento de BBB﻿ no pertenezca a AAA﻿.

Si AAA﻿ y BBB﻿ son dos conjuntos cuyos elementos son idénticos, escribimos A=B\bm{\color{c77b46}A = B}A=B﻿. A=BA = BA=B﻿ si simultáneamente vale que A⊆BA \subseteq BA⊆B﻿ y B⊆AB \subseteq AB⊆A﻿.

El conjunto que no posee ningún elemento se llama conjunto vacıˊo\small \color{c77b46} \textsf{\textbf{conjunto vacío}}conjunto vacıˊo﻿ y se nota ∅\bm{\color{c77b46} \empty}∅﻿.

1.1.2 ¿Coˊmo describir un conjunto?\textsf{\textbf{\color{c77b46}{1.1.2} \color{#d4d2cc}{¿Cómo describir un conjunto?}}}1.1.2 ¿Coˊmo describir un conjunto?﻿

Descripcioˊn por extensioˊn:\small \color{c77b46} \textsf{\textbf{Descripción por extensión:}}Descripcioˊn por extensioˊn:﻿ describir a un conjunto escribiendo sus elementos en una lista entre {}\{\}{}﻿, si éste tiene pocos elementos. Ejemplo:\small \color{c77b46} \textsf{\textit{Ejemplo:}}Ejemplo:﻿ X={1, @, manzana, naranja},Y={manzana, naranja}X = \{ \text{1, @, manzana, naranja} \}, Y = \{ \text{manzana, naranja} \}X={1, @, manzana, naranja},Y={manzana, naranja}﻿.

Descripcioˊn por comprensioˊn:\small \color{c77b46} \textsf{\textbf{Descripción por comprensión:}}Descripcioˊn por comprensioˊn:﻿ describir a un conjunto haciendo referencia a las características que comparten sus elementos. Ejemplo:\small \color{c77b46} \textsf{\textit{Ejemplo:}}Ejemplo:﻿ Y={x∈X:x es una fruta}Y = \{ x \in X : x \text{ es una fruta} \}Y={x∈X:x es una fruta}﻿. Se lee: Y es el conjunto de los x pertenecientes a x tales que x es una frutaY \text{ es el conjunto de los } x \text{ pertenecientes a } x \text{ tales que } x \text{ es una fruta}Y es el conjunto de los x pertenecientes a x tales que x es una fruta﻿ .

Generalmente, el conjunto referencial es R\realsR﻿. El universo en donde se trabaja generalmente se especifica, y se llama conjunto universal\small \color{c77b46} \textsf{\textbf{conjunto universal}}conjunto universal﻿ (U)(\mathcal{U})(U)﻿. Usualmente se trabaja con los siguientes conjuntos de números:

  • N\NN﻿, los naturales
  • Z\ZZ﻿, los enteros
  • Q\mathbb{Q}Q﻿, los racionales
  • R\realsR﻿, los reales
  • C\ComplexC﻿, los complejos

1.1.3 Subconjuntos del plano y el espacio\textsf{\textbf{\color{c77b46}{1.1.3} \color{#d4d2cc}{Subconjuntos del plano y el espacio}}}1.1.3 Subconjuntos del plano y el espacio﻿

Se conoce como plano\small \color{c77b46} \textsf{\textbf{plano}}plano﻿ al conjunto de los pares ordenados de números reales (x,y)(x, y)(x,y)﻿ y se nota R2\reals^2R2﻿. Se escribe: R2={(x,y):x,y∈R}\reals^2 = \{ (x, y) : x, y \in \reals \}R2={(x,y):x,y∈R}﻿. Su nombre surge del hecho que cualquier elemento de R2\reals^2R2﻿ puede representarse como un punto en el esquema de ejes cartesianos (xxx﻿ indica desplazamiento en el eje horizontal, e yyy﻿, en el vertical). El punto (0,0)∈R2(0, 0) \in \reals^2(0,0)∈R2﻿ se llama el origen de coordenadas.

Se conoce como espacio\small \color{c77b46} \textsf{\textbf{espacio}}espacio﻿ al conjunto de las ternas ordenadas de números reales (x,y,z)(x, y, z)(x,y,z)﻿ y se nota R3\reals^3R3﻿. Se escribe R3={(x,y,z):x,y,z∈R}\reals^3 = \{ (x, y, z) : x, y, z \in \reals \}R3={(x,y,z):x,y,z∈R}﻿. Su nombre se identifica con un ambiente tridimensional, y puede representarse en el espacio de tres ejes cartesianos. En este caso, (0,0,0)(0, 0, 0)(0,0,0)﻿ es el origen de coordenadas.

Generalizando estas ideas, se puede considerar al conjunto de tiras ordenadas de números reales Rn={(x1,…,xn):x1,…,xn∈R}\reals^n = \{ (x_1, …, x_n) : x_1, …, x_n \in \reals \}Rn={(x1​,…,xn​):x1​,…,xn​∈R}﻿, y se la llama n-upla ordenada de nuˊmeros reales\small\color{c77b46} \bm{n} \textsf{\textbf{-upla ordenada de números reales}}n-upla ordenada de nuˊmeros reales﻿. Para n>3n > 3n>3﻿, ya no es posible representarlo en un esquema de ejes cartesianos. El espacio Rn\reals^nRn﻿ se lo conoce como el espacio n-dimensional\small\color{c77b46} \textsf{\textbf{espacio }} \bm{n} \textsf{\textbf{-dimensional}}espacio n-dimensional﻿. La nnn﻿-upla (0,…,0)(0, …, 0)(0,…,0)﻿ de nnn﻿ ceros es el origen de coordenadas\small \color{c77b46} \textsf{\textbf{origen de coordenadas}}origen de coordenadas﻿.

1.2 Vectores de Rn\textsf{\textbf{\color{c77b46}{1.2} \color{#d4d2cc}{Vectores de }}} \bm{\reals^n}1.2 Vectores de Rn﻿

1.2.1 La nocioˊn de vector\textsf{\textbf{\color{c77b46}{1.2.1} \color{#d4d2cc}{La noción de vector}}}1.2.1 La nocioˊn de vector﻿

El segmento orientado de origen AAA﻿ y extremo BBB﻿ se llama vector ABundefined\color{c77b46} \textsf{\small\textbf{vector }} \bm{\color{c77b46} \overrightarrow{AB}}vector AB﻿ y sus elementos son:

  • La direccioˊn\small \color{c77b46} \textsf{\textbf{dirección}}direccioˊn﻿ (recta que contiene a los puntos AAA﻿ y BBB﻿),
  • El sentido\small \color{c77b46} \textsf{\textbf{sentido}}sentido﻿ (desde AAA﻿ hacia BBB﻿),
  • El moˊdulo\small \color{c77b46} \textsf{\textbf{módulo}}moˊdulo﻿ (longitud, distancia entre los puntos AAA﻿ y BBB﻿).

Estos elementos son propiedades intrínsecas de un vector y, por lo tanto, son independientes de su punto de origen. En cambio, el origen\small \color{c77b46} \textsf{\textbf{origen}}origen﻿ y el extremo\small \color{c77b46} \textsf{\textbf{extremo}}extremo﻿ de un vector dependen de su posición relativa con respecto a un sistema de coordenadas cartesianas. Por esto, consideramos que todos los vectores de igual dirección, sentido y módulo son equivalentes.

Todos los vectores de esta materia tienen su origen en el origen de coordenadas (0,…,0)∈Rn(0, …, 0) \in \reals^n(0,…,0)∈Rn﻿. Es decir que al indicar el vector v⃗=(x1,…,xn)∈Rn\vec{v} = (x_1, …, x_n) \in \reals^nv=(x1​,…,xn​)∈Rn﻿, nos referimos al vector con origen en el punto (0,…,0)∈Rn(0, …, 0) \in \reals^n(0,…,0)∈Rn﻿ y extremo en el punto (x1,…,xn)∈Rn(x_1, …, x_n) \in \reals^n(x1​,…,xn​)∈Rn﻿.

El origen de coordenadas también da lugar al vector nulo\small \color{c77b46} \textsf{\textbf{vector nulo}}vector nulo﻿ (no tiene longitud, ni dirección, ni sentido) y se nota 0⃗\color{c77b46} \bm{\vec{0}}0﻿.

1.2.2 Vectores en el espacio n-dimensional\textsf{\textbf{\color{c77b46}{1.2.2} \color{#d4d2cc}{Vectores en el espacio }}} \bm{n} \textsf{\color{#d4d2cc}\textbf{-dimensional}}1.2.2 Vectores en el espacio n-dimensional﻿

Dados dos vectores v⃗,w⃗∈Rn\vec{v}, \vec{w} \in \reals^nv,w∈Rn﻿, se define otro vector v⃗+w⃗∈Rn\vec{v} + \vec{w} \in \reals^nv+w∈Rn﻿. Este es llamado la suma de v⃗ y w⃗\color{c77b46} {\small\textsf{\textbf{suma de }}} \bm{\vec{v}} {\small\textsf{\textbf{ y }}} \bm{\vec{w}}suma de v y w﻿ (o vector suma\small \color{c77b46} \textsf{\textbf{vector suma}}vector suma﻿). Si v⃗=(x1,x2,...,xn)\vec{v} = (x_1, x_2, ..., x_n)v=(x1​,x2​,...,xn​)﻿ y w⃗=(y1,y2,...,yn)\vec{w} = (y_1, y_2, ..., y_n)w=(y1​,y2​,...,yn​)﻿ entonces: v⃗+w⃗=(x1+y1,x2+y2,…,xn+yn)\vec{v} + \vec{w} = (x_1 + y_1, x_2 + y_2, …, x_n + y_n)v+w=(x1​+y1​,x2​+y2​,…,xn​+yn​)﻿.

Gráficamente, el vector suma es el vector cuyo extremo es el punto al que se llega partiendo de (0,0)(0, 0)(0,0)﻿ al recorrer primero al vector v⃗\vec{v}v﻿ y, luego, desde el extremo de v⃗\vec{v}v﻿, el vector w⃗\vec{w}w﻿, o viceversa. Esto se conoce como la regla del paralelogramo\small \color{c77b46} \textsf{\textbf{regla del paralelogramo}}regla del paralelogramo﻿, dado que el vector suma coincide con la diagonal del paralelogramo de lados v⃗\vec{v}v﻿ y w⃗\vec{w}w﻿.

Ejemplo:\small \color{c77b46} \textsf{\textit{Ejemplo:}}Ejemplo:﻿ Si v⃗=(1,4)\vec{v} = (1, 4)v=(1,4)﻿ y w⃗=(3,1)\vec{w} = (3, 1)w=(3,1)﻿, entonces: v⃗+w⃗=(1,4)+(3,1)=(1+3,4+1)=(4,5)\vec{v} + \vec{w} = (1, 4) + (3, 1) = (1+3, 4+1)=(4, 5)v+w=(1,4)+(3,1)=(1+3,4+1)=(4,5)﻿.

Es posible también realizar el producto de un vector por un escalar\small \color{c77b46} \textsf{\textbf{producto de un vector por un escalar}}producto de un vector por un escalar﻿, alargando, acortando, o invirtiéndolo. Si v⃗=(x1,x2,…,xn)\vec{v} = (x_1, x_2, …, x_n)v=(x1​,x2​,…,xn​)﻿ y λ∈R\lambda \in \realsλ∈R﻿ entonces el producto del vector v⃗\vec{v}v﻿ por el escalar λ\lambdaλ﻿ es el vector λv⃗=(λx1,λx2,…,λxn)\lambda\vec{v} = (\lambda x_1, \lambda x_2, …, \lambda x_n)λv=(λx1​,λx2​,…,λxn​)﻿.

En la teoría de vectores, a los números reales se los conoce como escalares. El efecto de multiplicar un vector por −1-1−1﻿ invierte el sentido del vector. Se nota −v⃗-\vec{v}−v﻿ en lugar de (−1)v⃗(-1)\vec{v}(−1)v﻿.

Ejemplo:\small \color{c77b46} \textsf{\textit{Ejemplo:}}Ejemplo:﻿ Si v⃗=(2,1)\vec{v} = (2, 1)v=(2,1)﻿ y λ=−2\lambda = -2λ=−2﻿, entonces: λv⃗=−2(2,1)=(−2⋅2,−2⋅1)=(−4,−2)\lambda \vec{v} = -2(2, 1) = (-2\cdot 2, -2 \cdot 1) = (-4, -2)λv=−2(2,1)=(−2⋅2,−2⋅1)=(−4,−2)﻿.

Estas operaciones tienen las siguientes propiedades:

  1. v⃗+(w⃗+u⃗)=(v⃗+w⃗)+u⃗\vec{v} + (\vec{w} + \vec{u}) = (\vec{v} + \vec{w}) + \vec{u}v+(w+u)=(v+w)+u﻿
  1. v⃗+w⃗=w⃗+v⃗\vec{v} + \vec{w} = \vec{w} + \vec{v}v+w=w+v﻿
  1. Si λ∈R,λ(v⃗+w⃗)=λv⃗+λw⃗\lambda \in \reals, \lambda(\vec{v} + \vec{w}) = \lambda \vec{v} + \lambda \vec{w}λ∈R,λ(v+w)=λv+λw﻿
  1. Si λ1,λ2∈R\lambda_1, \lambda_2 \in \realsλ1​,λ2​∈R﻿, (λ1+λ2)v⃗=λ1v⃗+λ2v⃗(\lambda_1 + \lambda_2) \vec{v} = \lambda_1 \vec{v} + \lambda_2 \vec{v}(λ1​+λ2​)v=λ1​v+λ2​v﻿ y (λ1λ2)v⃗=λ1(λ2v⃗)(\lambda_1 \lambda_2) \vec{v} = \lambda_1 (\lambda_2 \vec{v})(λ1​λ2​)v=λ1​(λ2​v)﻿
  1. 0⃗+v⃗=v⃗\vec{0} + \vec{v} = \vec{v}0+v=v﻿
  1. 1⋅v⃗=v⃗1 \cdot \vec{v} = \vec{v}1⋅v=v﻿
  1. v⃗+(−1)v⃗=0⃗\vec{v} + (-1)\vec{v} = \vec{0}v+(−1)v=0﻿
  1. 0⋅v⃗=0⃗0 \cdot \vec{v} = \vec{0}0⋅v=0﻿

Gráficamente, para averiguar v⃗−w⃗\vec{v} - \vec{w}v−w﻿ primero se calcula −w⃗-\vec{w}−w﻿ (invirtiendo su sentido) y, luego, se suma v⃗\vec{v}v﻿ con −w⃗-\vec{w}−w﻿ usando la regla del paralelogramo.
